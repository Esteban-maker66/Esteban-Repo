# 🌊 Arrecife Digital - Biblioteca Virtual

Una plataforma moderna y responsiva para la gestión de recursos educativos, diseñada para la comunidad del **Mar Azul**.

## Características
- **Glassmorphism UI:** Interfaz elegante con transparencias y desenfoque de fondo.
- **Buscador Inteligente:** Filtrado en tiempo real que optimiza el espacio de trabajo.
- **Sistema de Favoritos:** Persistencia de datos utilizando **Supabase**.
- **Panel de Administración:** Gestión de aprobación de nuevos recursos con modales personalizados.
- **Optimizado para PC:** Navegación horizontal mediante `Shift + Scroll`.

## Tech Stack
- **Frontend:** HTML5, CSS3 (Variables modernas, Flexbox, Grid), JavaScript (ES6+).
- **Backend as a Service:** Supabase (Auth & Database).
- **Hosting:** Vercel.
